# Ziyu Wang Academic Portfolio

Static, single-page academic portfolio for Wang Ziyu (王子煜), PhD researcher in autonomous driving at Jilin University.

Open `index.html` directly in a browser. No build step, package manager, or local server is required.

## Deployment

1. `git init && git add . && git commit -m "initial"`
2. Create GitHub repo named `YOUR_USERNAME.github.io` or any repository name.
3. `git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git`
4. `git push -u origin main`
5. GitHub Actions auto-deploys, then visit `https://YOUR_USERNAME.github.io`

## Formspree Contact Form

1. Sign up free at `https://formspree.io`
2. Create a new form and copy the form ID.
3. Replace `YOUR_FORM_ID` in `main.js`.

## Placeholders

- Replace `YOUR_USERNAME` references with your actual GitHub username when publishing.
- Replace `assets/avatar.jpg` with a real profile photo when available.
- Replace project GitHub links currently set to `#`.
- Replace the Google Scholar placeholder URL in `index.html`.
